史诗级游戏
俄罗斯方块 (Tetris)
游戏简介
俄罗斯方块是由俄罗斯程序员阿列克谢·帕吉特诺夫于1984年开发的经典益智游戏。游戏目标是通过移动和旋转下落的方块，使其完整地填满水平行。当一行被填满时，该行会消除并得分。本游戏使用Python和Pygame库开发，完美还原了经典俄罗斯方块的玩法和体验。
游戏特色
•经典玩法：完美还原传统俄罗斯方块的游戏机制
•精美画面：采用现代配色方案和3D立体方块效果
•方块预览：右侧面板实时显示下一个方块
•下落投影：显示方块落地的参考位置
•等级系统：随着消除行数增加，等级和难度逐步提升
•触底锁定延迟：方块触底后有短暂延迟，增加操作空间
•墙踢系统：旋转时自动尝试偏移，解决靠墙旋转问题
方块类型
游戏包含7种经典方块类型，每种都有独特的颜色和旋转形态：
方块	形状	特点
I型 (青色)	长条形	可横放或竖放，适合填满空隙
O型 (黄色)	正方形	无法旋转，位置摆放较简单
T型 (紫色)	T字形	最常用的方块，旋转后形态变化多
S型 (绿色)	S字形	与Z型互为镜像
Z型 (红色)	Z字形	与S型互为镜像，需要注意旋转方向
J型 (蓝色)	J字形	与L型互为镜像
L型 (橙色)	L字形	与J型互为镜像
操作说明
游戏支持键盘操作，具体按键如下：
基础操作
1.← / →：左右移动方块
2.↑：顺时针旋转方块
3.↓：加速下落（每下移一行得1分）
4.空格键：硬降，直接落到底部（每下落一格得2分）
游戏控制
5.P：暂停/继续游戏
6.R：游戏结束后重新开始
7.ESC：退出游戏
计分规则
游戏根据消除的行数进行计分：
消除行数	得分
单行 (Single)	100 分
双行 (Double)	300 分
三行 (Triple)	500 分
四行 (Tetris)	800 分
等级系统
游戏每消除10行提升一个等级。等级越高，方块下落速度越快，游戏难度越大。初始下落间隔为500毫秒，每升一级减少50毫秒，最快为100毫秒。
安装与运行
环境要求
•Python 3.x
•Pygame 库
运行步骤
1. 安装依赖：
pip install pygame
2. 运行游戏：
python tetris.py
游戏目标
尽可能长时间地保持游戏，通过消除行数获取高分。合理利用方块预览和下落投影功能，提前规划方块的摆放位置，尽可能一次消除更多行以获得更高分数。祝你玩得愉快！
